# 每次心跳时检查以下任务，按优先级执行

## 📌 每日盘前情报（优先级最高）
触发条件：交易日 07:00-09:30
任务：
1. 扫描全球情报（美伊战争进展、隔夜美股、央行动态）
2. 扫描A股重大消息
3. 检查持仓止损止盈状态
4. 如有紧急情报，立即推送用户

## 📌 持仓监控（优先级高）
触发条件：交易日 09:30-15:00
任务：
1. 检查持仓盈亏
2. 用10维度框架分析持仓健康度：
   - 基本面：公司业务、行业地位、护城河
   - 政策：行业政策、监管风险、政策红利
   - 周期：行业周期、产品价格周期、经济周期
   - 赚钱能力：ROE、毛利率、现金流、盈利稳定性
   - 估值：PE/PB相对历史、相对同行
   - 管理层：创始人背景、治理结构、股权激励
   - 竞争格局：行业集中度、护城河、竞争对手
   - 财务健康：负债率、现金流、应收账款
   - 催化剂：未来可能触发股价上涨的事件
   - 风险点：最大的不确定性因素
3. 如果出现基本面恶化/政策重大利空/周期拐点/财务风险 → 推送预警
4. 不再使用固定百分比止损，改用逻辑止损

## 📌 收盘复盘（优先级中）
触发条件：交易日 15:30-16:30
任务：
1. 检查当日持仓盈亏
2. 扫描盘后重大消息
3. 更新次日关注清单

## ⚠️ 紧急情报（随时）
触发条件：任何时间发现以下情报
- 战争升级/停火
- 央行意外加息/降息
- 持仓股突发利空/利好
- 市场熔断/异常波动
- 重大政策出台

## 📌 非交易日/盘后
触发条件：交易时段外
任务：HEARTBEAT_OK
