## 持仓分析框架升级

### 10维度分析框架

| 维度 | 说明 | 为什么重要 |
|------|------|-----------|
| 基本面 | 公司业务、行业地位、护城河 | 决定公司长期价值 |
| 政策 | 行业政策、监管风险、政策红利 | 政策能成就一家公司，也能毁掉一家公司 |
| 周期 | 行业周期、产品价格周期、经济周期 | 决定买入和卖出时机 |
| 赚钱能力 | ROE、毛利率、现金流、盈利稳定性 | 能赚钱的公司才能穿越周期 |
| 估值 | PE/PB相对历史、相对同行 | 决定买入时机，好公司买贵了也亏钱 |
| 管理层 | 创始人背景、治理结构、股权激励 | 好管理层能带公司穿越周期 |
| 竞争格局 | 行业集中度、护城河、竞争对手 | 决定定价权和长期盈利能力 |
| 财务健康 | 负债率、现金流、应收账款 | 防止踩雷财务造假或资金链断裂 |
| 催化剂 | 未来可能触发股价上涨的事件 | 决定持有周期和卖出时机 |
| 风险点 | 最大的不确定性因素 | 知道最坏情况是什么 |

### 关键教训

- 不再使用固定百分比止损（如-8%）
- 改用逻辑止损：基本面恶化/政策重大利空/周期拐点/财务风险
- 三只持仓都是好公司，没有卖出理由
